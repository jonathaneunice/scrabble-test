
Example functions for Scrabble-like game.

Dictionary used from 
https://github.com/jonbcard/scrabble-bot

Given to me as an interactive coding test.  First commit is what I
arrived at in real-time.

Second commit (and subsequent ones, if any) is what I cleaned the
code into with a few more minutes' work.

It was an interesting exercise.  "Here, do this simple thing" not
always so simple when you're being evaluated (and possibly, timed).
Also, what seem like simple exercises can call on diverse skills,
including data structures, order statistics, combinatorics, module 
knowledge, code style and clarity, debugging...
